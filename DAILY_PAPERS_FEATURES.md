## 🚀 如何使用

### 1. 访问每日更新页面
在浏览器中访问：`http://localhost:8080/daily`

### 2. 选择感兴趣的话题
点击顶部的话题标签（大模型、AI、工程等）

### 3. 浏览论文
查看最新论文列表，包含标题、作者、摘要等信息

### 4. AI 深度解读
点击论文卡片上的"**AI解读**"按钮，智谱 AI 会生成：
- 中文摘要
- 5个核心要点
- 创新点
- 方法论
- 主要结果
- 局限性
- 未来方向
- 阅读难度
- 适合人群
- 中文标签

### 5. 查看完整论文
点击"查看"按钮跳转到论文详情页，可以在线阅读 PDF

## 🔧 定时更新设置

### 方式一：手动更新
```bash
cd backend
python -m app.tasks.daily_paper_updater
```

### 方式二：使用 cron（推荐）
```bash
crontab -e

# 添加以下行（每天凌晨2点更新）
0 2 * * * cd /path/to/backend && python -m app.tasks.daily_paper_updater >> /var/log/daily_papers.log 2>&1
```

## 🎉 功能亮点

1. **完全自动化**：每天自动从 arXiv 获取最新论文
2. **AI 深度解读**：智谱 GLM-4.7-Flash 提供专业的论文分析
3. **中文友好**：专为中文用户优化，提供中文摘要和解读
